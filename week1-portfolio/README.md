# Week 1 Portfolio

This is a basic portfolio website structure for Week 1.

## Structure

- `index.html`: Main HTML file
- `css/`: Stylesheets
  - `style.css`: Main styles
  - `responsive.css`: Responsive styles
  - `variables.css`: CSS variables
- `js/`: JavaScript files
  - `navigation.js`: Navigation script
- `images/`: Image files
  - `profile.jpg`: Profile image
  - `project1.jpg`: Project image
  - `icons/`: Icon files
- `README.md`: This file
- `.gitignore`: Git ignore file
